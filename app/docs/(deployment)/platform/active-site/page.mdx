# Active Site

**Active Site** คือหน้าสำคัญของ Platform ที่ใช้สำหรับแสดงสถานะของระบบหรือเว็บไซต์ต่าง ๆ  
ที่ถูก Deploy และกำลังทำงานอยู่บน Server ผ่านระบบ Container (เช่น Docker)

หน้านี้ถูกออกแบบมาเพื่อให้ผู้ใช้งานสามารถ **ตรวจสอบสถานะการทำงานของระบบได้แบบ Real-time**  
โดยไม่จำเป็นต้องเข้าถึง Server โดยตรงผ่าน Command Line

---

## Purpose of Active Site

หน้า Active Site มีวัตถุประสงค์หลักดังนี้

- แสดงรายการเว็บไซต์หรือแอปพลิเคชันที่ถูก Deploy แล้ว
- ตรวจสอบสถานะการทำงานของแต่ละ Container (เช่น Running / Stopped)
- ช่วยให้ผู้ดูแลระบบและผู้ใช้งานสามารถติดตามระบบได้จาก Web UI
- ลดความซับซ้อนในการบริหารจัดการระบบภายในเครือข่าย (LAN)
