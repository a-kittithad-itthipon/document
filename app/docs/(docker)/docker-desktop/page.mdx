เพื่อไม่ให้เปลืองทรัพยากรเครื่อง เมื่อทดลองเสร็จแล้วควรปิดทุกครั้ง

### ขั้นตอนการ Build Image
เมื่อคุณมีไฟล์ `Dockerfile` ในเครื่องแล้ว ให้ใช้คำสั่งผ่าน Terminal หรือ Docker Desktop:

**รูปแบบคำสั่ง:**
`docker build -t <image-name>:<tag> <path>`

* **ตัวอย่าง:** `docker build -t my-app:v1 .`
* **คำอธิบาย:** สร้าง Image ชื่อ `my-app` เวอร์ชัน `v1` โดยใช้ไฟล์ Dockerfile ที่อยู่ในโฟลเดอร์ปัจจุบัน (`.`)

### การแบ่งปัน Image (Docker Hub)
1. **Tag**: ตั้งชื่อให้ตรงกับ Repository บน Cloud
   `docker tag <image-name> <username>/<repository>:<tag>`
2. **Push**: อัปโหลดขึ้น Docker Hub
   `docker push <username>/<repository>:<tag>`
