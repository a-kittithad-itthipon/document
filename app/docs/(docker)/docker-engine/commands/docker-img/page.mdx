# Docker Image

## What is Docker Image

**Docker Image** เปรียบเสมือน "แม่พิมพ์" หรือ "พิมพ์เขียว" (Template) แบบ Read-only ที่บรรจุ Source Code, Libraries, Dependencies, เครื่องมือ และไฟล์ต่างๆ ที่จำเป็นสำหรับการรันแอปพลิเคชันเอาไว้ภายใน Image หนึ่งตัวจะถูกนำไปสร้างเป็น Container ได้หลายตัว โดย Image มักจะถูกสร้างขึ้นมาจากคำสั่ง `docker build` หรือดึงมาจาก Registry กลางอย่าง Docker Hub


## Basic Commands      

### Image (Search)

ค้นหารูปภาพที่มีอยู่บน Docker Hub ได้โดยใช้ **docker** คำสั่งพร้อม **search** คำสั่งย่อย 

```bash
docker search ubuntu
```
 > *ระบบจะแสดงรายชื่อ Image ที่เกี่ยวข้อง พร้อมรายละเอียด เช่น ความนิยม (Stars) และสถานะ Official*

ผลลัพธ์ที่ได้จะมีลักษณะคล้ายกับตัวอย่างต่อไปนี้:
```bash

open-liberty                     Open Liberty multi-architecture images based…   51        [OK]

```


### Image (Pull)


ดาวน์โหลด Image จาก Docker Hub มาเก็บไว้ในเครื่อง (ในตัวอย่างคือการดึง Image ของ Ubuntu)

```bash
docker pull ubuntu
```

จะเห็นผลลัพธ์ดังต่อไปนี้:
```bash
Using default tag: latest
latest: Pulling from library/ubuntu
e0b25ef51634: Pull complete
Digest: sha256:9101220a875cee98b016668342c489ff0674f247f6ca20dfc91b91c0f28581ae
Status: Downloaded newer image for ubuntu:latest
docker.io/library/ubuntu:latest
```

### Image (List)
ตรวจสอบว่าในเครื่องของเรามี Image อะไรถูกดาวน์โหลดเก็บไว้บ้าง

```bash
docker images
```
 > *ระบบจะแสดงตารางข้อมูลประกอบด้วย Repository, Tag, Image ID, วันที่สร้าง และขนาดไฟล์*

ผลลัพธ์ที่ได้จะมีลักษณะคล้ายกับตัวอย่างต่อไปนี้:
```bash
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
ubuntu              latest              1d622ef86b13        3 weeks ago         73.9MB
hello-world         latest              bf756fb1ae65        4 months ago        13.3kB
```

### Image (Remove Image)
ลบ Image ที่ไม่ต้องการออกจากเครื่อง โดยระบุ Image ID หรือชื่อ Image (ตรวจสอบว่าไม่มี Container ไหนใช้งาน Image นี้อยู่)

```bash
docker rmi nginx
```

ผลลัพธ์ที่ได้จะมีลักษณะคล้ายกับตัวอย่างต่อไปนี้:
```bash
Untagged: nginx:latest
Untagged: nginx@sha256:bb129a712c24...
Deleted: sha256:bb129a712c24...
```
